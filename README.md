# 躲猫猫 HideCatCat

FFXIV 多人躲猫猫小游戏 Dalamud 插件。

## 玩法

- **猫队**：在限定时间内抓住所有鼠队玩家即可获胜
- **鼠队**：躲避猫队的追捕，存活到时间结束即可获胜
- 猫队靠近鼠队 5 码以内即判定抓住
- 鼠队超出边界范围 5 秒自动判负
- 猫队游戏中会隐藏所有其他玩家的头顶名牌，鼠队靠听脚步和观察环境躲避

## 功能

- **边界蒙版**：鼠队距边界 10 yalms 触发红色径向渐变蒙版，越靠近边界越浓
- **距离提示**：插件界面实时显示距边界剩余距离
- **服务器地址**：下拉框选择，默认「默认躲猫猫服务器地址」，自定义地址自动持久化历史
